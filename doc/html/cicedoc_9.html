Numerical implementation

CICE is written in FORTRAN90 and runs on UNIX host platforms; it has been developed on an SGI Origin 2000 at Los Alamos National Laboratory. The code is parallelized via grid decomposition with MPI; an effort is underway to develop an efficient hybrid parallelization that utilizes both OpenMP and MPI for better load balancing across processors.

A second, "external" layer of parallelization involves message passing between CICE and the flux coupler, which may be running on different machines in a distributed system. The parallelization scheme for CICE was designed so that MPI could be used for the coupling along with either OpenMP or MPI (or neither) internally. The internal parallelization method is set at compile time with the BINTYPE definition in the make scripts. Message passing between the ice model and the flux coupler is accomplished with MPI, regardless of the type of internal parallelization used for CICE. OpenMP directives have been removed from this version of CICE while we develop a hybrid scheme.

Directory structure

The present code distribution includes a run script and some input files. The user may wish to alter paths in the run script; other parameters that may be changed at run time are found in ice_in.
GNUmakefile
primary makefile
IRIX64.defs
macro definitions for SGI IRIX, used by GNUmakefile
Makedepends.pl
perl script that determines module dependencies
